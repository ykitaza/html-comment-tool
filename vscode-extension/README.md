# html-comment: Review for AI (VS Code extension)

任意のファイルをエクスプローラーで右クリック →「html-comment: Review for AI」で、
横に Webview レビューパネルを開きます。行（範囲）にコメントを付け、
AI 向けの修正プロンプトをコピーできます（ブラウザ版 html-comment と同じ UI）。

- コメントはワークスペースに保存（再オープンで復元）
- パネルのコメントをクリックすると実エディタの該当行へジャンプ
- プロンプトのテンプレート（修正/質問/レビュー/カスタム）を設定で選択

v1 はソースレビュー（行コメント）。HTML/Markdown のプレビュー描画は今後対応予定。
